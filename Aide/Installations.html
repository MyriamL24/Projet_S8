Installations

Stat'nDat peut s'exécuter sur Linux, Windows ou Mac en 64 bits ou 32 bits, à condition que votre environnement intègre les langages nécessaires. Ainsi, Stat'nDat s'exécute avec Python 2.7, utilise le langage Latex et nécessite une base de données gérée par PostgreSQL.

1) Modules complémentaires :

Voici les différents modules et bibliothèques Python qui sont importés dans Stat'nDat. Pensez à vérifier avant d'exécuter l'application, la présence de chacun d'entre eux sur l'environnement que vous utilisez.

  • Tkinter
  • tkMessageBox
  • anydbm
  • Pmw
  • ttk
  • os
  • PIL
  • matplotlib
  • scipy
  • psycopg2
  • pickle
  • time
Pmw

C'est la bibliothèque Python qui sert à afficher tous les widgets de l'interface. Nous l'avons fournie dans le dossier de l'application : "python-pmw_1.3.2.orig.tar.gz". Pour l'installer sous Linux ou MacOS, décompressez le dossier puis tapez cette commande dans votre console : sudo python setu.py install

Matplotlib

Pour afficher les graphiques. Cette bibliothèque n'est pas fournie par défaut, avec un OS Linux, la commande "sudo apt-get install matplotlib" suffit. En revanche, sur un système windows, vous pourrez trouver le fichier d'exécution ici :

Scipy

Cette bibliothèque non fournie par défaut est utilisée par notre application pour générer les calculs statistiques. Tout comme pour Matplotlib, Linux permet directe en utilisant la commande "sudo apt-get install scipy"

2) Dossiers et fichiers :

Stat'nDat comprend trois dossiers :

  • Images
    • Icones
    • Graphiques
  • Rapport
  • Donnees
Merci de bien vouloir vérifier la présence de ces dossiers avant toute exécution afin d'éviter les rapports d'erreur. Par ailleurs, le dossier Rapport doit obligatoire contenir un fichier .tex nommé "file.tex".
